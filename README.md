
Pac-Man Game (Java)

Description:

 The game simulates the movement of Pac-Man and a ghost inside a grid. The player controls Pac-Man using keyboard inputs, collects dots to increase score, and avoids the ghost.

 Features:

Grid-based game environment
 Pac-Man movement (up, down, left, right)
Random ghost movement
 Score tracking system
 Collision detection (game over when ghost catches Pac-Man)
 Win condition when all dots are collected

 Game Controls:

 Key  Action     
 
 W :   Move Up    
 S :   Move Down  
 A :   Move Left  
 D :   Move Right 

 Steps Taken:

1. Created a pacman class to manage game logic
2. Defined a 2D grid using a character array
3. Added Pac-Man and ghost positions using row and column variables
4. Implemented movement using an enum for directions
5. Developed Pac-Man movement with obstacle and dot checking
6. Implemented random ghost movement using Java Random class
7. Added score system when Pac-Man collects dots (`.`)
8. Created a method to display the grid in console
9. Implemented game loop using Scanner for user input
10. Added win and lose conditions


How to Run the Project

1. Open the project in any Java IDE (IntelliJ, Eclipse, or VS Code)
2. Make sure Java is installed on your system
3. Compile the program
4. Run the `Main` class
5. Use keyboard inputs (W, A, S, D) to play the game


 Output Example

* The grid is displayed in the console
* `P` represents Pac-Man
* `G` represents the ghost
* `.` represents dots to collect
* `#` represents walls



 Conclusion

This project demonstrates the use of object-oriented programming concepts, arrays, enums, loops, and user input handling in Java. It provides a basic understanding of how simple games can be developed using core programming concepts.
